# Smart Step Counter - ESP32 Project

Podomètre connecté avec double connectivité BLE/MQTT pour applications mobiles et web.

##  Fonctionnalités

**Version BLE (Mobile) :**
- Détection de pas en temps réel
- Connexion directe smartphone via Bluetooth
- Service BLE avec 5 caractéristiques
- Autonomie : 5 jours (2000mAh)

**Version MQTT (Web) :**
- Streaming données 20Hz vers cloud
- Dashboard web avec graphiques
- Historique hebdomadaire
- Support ML externe

##  Hardware

- ESP32-WROOM-32
- Accéléromètre MMA7660
- LED RGB
- Batterie Li-Po 2000mAh
